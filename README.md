# plane-war-vue3
基于vue3和canvas开发的飞机大战小游戏demo。
实现了自定义的 vue Renderer ，渲染使用 pixijs

# 游戏链接
https://wuyangqin.github.io/plane-war-dist/index.html

# 游戏操作
点击开始游戏，方向键控制飞机移动，按空格发射子弹
